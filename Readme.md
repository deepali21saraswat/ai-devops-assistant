# AI DevOps Assistant 🚀

This project is an AI-powered DevOps tool that analyzes logs and suggests fixes using LLMs.

## Goals
- Build AI-powered log analyzer
- Integrate with FastAPI
- Deploy using Docker & Kubernetes
- Use Argo CD for GitOps deployment

## Tech Stack
- Python
- FastAPI
- OpenAI API
- Docker
- Kubernetes
- Argo CD
